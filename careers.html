Unity Gameplay Programmer

💻 Engineering 🌍 Remote 📋 Freelance / Contract

About the Role

Join us in building the core gameplay systems for NoSaints. Our codebase uses a feature-oriented MVP + State Machine architecture powered by Zenject. You'll work on movement, combat, interaction, and narrative trigger systems — all designed for clean separation and fast iteration.

What You'll Do

  • Implement gameplay features following the MVP + State Machine pattern
  • Build and maintain ScriptableObject-based state definitions
  • Integrate with KINEMATION procedural animation systems
  • Write Physics abstractions and input command handlers
  • Collaborate on architecture decisions and code reviews

What We're Looking For

  • Strong C# and Unity experience (2+ years)
  • Familiarity with dependency injection (Zenject preferred)
  • Understanding of state machines, SOLID principles, and clean architecture
  • Experience with CharacterController or Rigidbody-based movement
  • Comfort with assembly definitions and namespace conventions
